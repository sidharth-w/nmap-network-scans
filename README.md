# Metasploitable2 Security Testing Lab

## Objective
To practice network scanning, enumeration, and basic exploitation in a safe lab environment.

## Environment
- Attacker: Kali Linux (VM)
- Target: Metasploitable2
- Virtualization: VirtualBox

## Tools Used
- Nmap
- Metasploit Framework
- Nikto
- Hydra

## Tasks Performed
- Network discovery and port scanning
- Service and version enumeration
- Vulnerability identification
- Password brute-force (lab only)

## Key Learnings
- Importance of open ports and weak credentials
- How attackers enumerate services
- Basics of exploitation workflow

⚠️ This project was performed in a legal lab environment for educational purposes only.
